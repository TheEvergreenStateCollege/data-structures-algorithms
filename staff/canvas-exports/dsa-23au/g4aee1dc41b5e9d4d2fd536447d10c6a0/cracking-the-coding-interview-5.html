At the end of class (3pm)

If you are making up this assignment, set aside a 2 hour period of time when you can work uninterrupted (with rest breaks).

Submit the following here on Canvas

  • link to your profile on HackerRank or other coding problem website
  • links to a specific problem you attempted or worked on *only* during this 2 hour period of time, with a submission
    • Click on the "Submissions" tab on the left.
    • Choose your submission from the list provided.
    • Copy the URL in the browser and submit it on Canvas.
      • It will look like
        • https://www.hackerrank.com/challenges/diagonal-difference/submissions/code/356718277
  • names of any teammates that you worked with
    • you are encouraged to pair program on these! It is more fun and motivating, but does take work to set up a time outside of class.
    • screenshare in a voice channel on Discord (one of the study rooms) is recommended
    • when doing this, one of you should type into their Hackerrank account and share their screen
    • the other should navigate
    • trade places every 15 minutes or so, whenever you get tired, and the previous driver will narrate the code they just typed (as the new navigator) for the other person, who is the new driver.
    • repeat until you've been practicing for two hours.
  • links to your search engine queries and GPT chats
    • it is okay to use information you find on Google, Stackoverflow, ChatGPT, etc. with the following conditions
      • you type in all the code yourself character-by-character (no copy and paste)
      • you can explain the purpose of each line of code
      • you know the correct order to type in the code, 3 lines of code / comments at a time, so that the code always compiles without errors, and that it leads to your desired submission.
  • describe the progress you made problems you worked on during classtime and any thoughts that come up.
  • if you have time left, repeat this with another problem of similar difficulty.
